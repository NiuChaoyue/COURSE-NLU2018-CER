1. **This is the source code for**

   my course project, titled Clinic Entitle Recognition (CER), in the graduate course Natural Language Understanding 2018.
   
   Based on https://github.com/guillaumegenthial/sequence_tagging 
   
2. **Online test results:**

   >link: http://bcmi.sjtu.edu.cn/~nlp4g2018_eval/
   
   >1st Test: accuracy: 93.61%; precision: 79.73%; recall: 81.45%; FB1: 80.59
   
   >2nd Test: accuracy: 93.39%; precision: 79.42%; recall: 81.06%; FB1: 80.23
   
   >**3nd Test (Best): accuracy: 93.96%; precision: 81.16%; recall: 82.38%; FB1: 81.76**
   
3. **Major Python3 libs:**

   tensorflow, numpy, logging

4. **Compile commands:**

   >run from start, please use
   >```sh
   >make glove
   >make run
   >```
   
   >run from train, please use
   >```sh
   >make run
   >```
   
   >run just test, please use
   >```sh
   >python test.py
   >```
